# Project Flight
Created Using C++, Anltr4, and LLVM

# Dependencies
- make
- cmake

## To get Started
1. Install make and cmake using your distro's package manager (`sudo apt install cmake make` for ubuntu/debian) 
2. Build the source using cmake by running `cmake` in the root directory of the cloned repository
3. Add an alias by running `alias flc="/usr/local/bin/flightcompiler"` and put the same command in your bash startup file (for example .bashrc)
4. Compile your code with `flc src.fly` and run the created executable file
